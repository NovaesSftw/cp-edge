# cp-edge

■ Vinheria Agnello – Sistema de
Monitoramento de Luminosidade
Este projeto apresenta o desenvolvimento de um sistema embarcado utilizando Arduino para
monitoramento da luminosidade em ambientes de armazenamento de vinhos. O objetivo é garantir
condições ideais de conservação, evitando degradações causadas pela luz.
■ Objetivos
- Monitorar luminosidade em tempo real
- Classificar níveis em OK, Alerta e Crítico
- Exibir status via LEDs
- Emitir alerta sonoro com buzzer
■■ Componentes
Arduino Uno
Sensor LDR
Resistores (10kΩ e 220Ω)
LEDs (verde, amarelo, vermelho)
Buzzer
Protoboard e jumpers
■ Lógica de Funcionamento
Nível Condição Ação
OK Baixa luminosidade LED Verde
Alerta Luminosidade média LED Amarelo
Crítico Alta luminosidade LED Vermelho + Buzzer
■■ Execução
Monte o circuito, carregue o código no Arduino e simule no Tinkercad. Ajuste a luminosidade para
observar os diferentes estados do sistema.
